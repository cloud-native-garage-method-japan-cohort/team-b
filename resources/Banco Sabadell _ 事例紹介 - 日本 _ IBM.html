Banco Sabadell

デジタル・バンキング・モデルのパイオニア IBM Cloudと Red Hatでセキュリティーを強化

メキシコ初の100%モバイル・バンクであるBanco Sabadell S.A.がメキシコでオンライン・バンキング事業に乗り出した時、そのITインフラストラクチャーの運用・保守を行っていたのはスペインの親会社でした。しかしセキュリティーおよびデータ保護に関するメキシコ国内の政策と規制要件を順守するために、Banco Sabadellの幹部はITインフラストラクチャーをメキシコ国内のクラウド・サービスに移行しました。

ビジネス上の課題

ここでBanco Sabadellは課題に直面しました。Banking as a service (BaaS)モデルの根幹となるクラウドネイティブ・アプリケーションは、マイクロサービスとDevOpsプロセスを使用して開発されていました。しかし、そのクラウド・サービス・プロバイダーはDevOps管理の専門知識に乏しく、新しいサービスやアプリを市場投入に後れを取っていました。さらに、クラウド・サービスそのものが停止し、顧客サービスに支障をきたす事態に陥りました。

解決策は、より信頼性の高いクラウド・テクノロジーと専門知識を備えた新しいテクノロジー・プロバイダーを採用し、環境を管理してソフトウェア開発を改善することでした。

概要と経緯

先進のインフラストラクチャーで事業戦略をサポート

BaaSモデルの利用と管理の促進を実現し、デジタル・ソリューションおよびサービスを革新し続けて、全メキシコ百万人のお客様に届けることを目標に、Banco SabadellはIBM Services®を採用しました。
Banco Sabadellは2018年にIBM Cloud®への移行に着手し、まず開発とテスト用に4つの仮想マシン(VM)と4つのベアメタル・サーバーにIBM Cloud®を採用しました。
2019年から2020年にかけてIBM Cloudの使用量を増やし、50を超えるVMを追加で展開し、スペインのビターウォーターにあるデータセンターからメキシコのIBM Cloudデータセンターにワークロードを移行しました。この環境ではVMware on IBM CloudサービスをベースとしたVMwareファームを使用し、俊敏かつ柔軟な移行が可能になります。このメキシコ国内のクラウドを利用することにより、セキュリティーおよびデータ保護に関するポリシーと規制要件に対応できるようになります。
Banco SabadellはアプリケーションをIBM Cloudに移行し、VMとIBM Cloudベアメタル・サーバーの両方にワークロードを展開しました。VMwareワークロードを移行したIBM Cloud for VMware Solutionsは、セキュリティー機能豊富な自動デプロイメント・アーキテクチャーで、金融機関でのコンプライアンス管理に役立ちます。
また、Red Hat® Enterprise Linux®で稼働するIBM Cloudサービスを活用することで、Banco Sabadellの銀行環境では柔軟性とスピードにより管理が効率的になり、顧客体験の向上が見込まれます。

「Bank as a serviceモデルのパイオニアであることは大きな課題を抱えることでもあります。これを解決するにはIBMの幅広い業界経験が必要でした」とは、Banco SabadellのCOO Fernando Boza氏の言葉です。
Banco Sabadellは、アプリケーションをモダナイズして革新的なサービスをお客様に提供するために、IBMパブリッククラウド上でインフラストラクチャーを展開し、Red Hat Enterprise Linuxを使用することにしました。
IBMチームは、モバイル・デバイスをマネージド・サービスで管理できるように、IBM® MaaS360®テクノロジーを使用してアプリケーション監視用のダッシュボードも作成し、新たに災害復旧(DR)サービスも実装しました。
開発側では、IBMソフトウェアの専門家が以前のプロバイダーのDevOps作業を引き継ぎ、新しい機能を追加しました。また、アジャイルな反復プロセスを考案してアプリケーションの開発、配信、統合が連続したサイクルとして実行できるようにしました。Kubernetesオーケストレーション・プラットフォームであるRed Hat OpenShiftとRed Hat Enterprise Linuxに基づいたオープン・ハイブリッドクラウド環境は、Containers as a service(CaaS)の基盤を提供します。
クラウド・コンピューティング・モデルのCaaSでは、開発者はオンプレミス・リソースおよびクラウドを使用し、コンテナ・ベースで抽象化してアプリケーションを展開および管理することで、要件を最適に満たすことができます。さらには、スケーリング、更新、およびその他の主要なIT機能を自動化することができます。CaaSなら、より効率的な最新のIT環境を提供できます。

Banco Sabadellの主要な事業戦略の1つであるBaaSモデルを、IBM API Connect®テクノロジーが実現します。このソリューションは銀行のAPIを統合し、ワークスペースを作成します。このワークスペースは、フィンテック企業や大企業がBanco Sabadellと連携して、新しいオンライン金融サービスやアプリを作成するための最初の窓口としての役割を果たします。
クラウド移行、マネージド・サービス、API管理、CaaSは、銀行のBaaSプラットフォームを改善するための基本的な要素であり、Banco Sabadellが競合他社と差別化する新しいアプリおよびサービス開発の鍵となります。
もう1つの重要な戦略は、IBM Cloud Pak® for Dataを使用してさまざまなソースからのデータを統合する「ゴールデン・ソース・リポジトリー」を構築することです。Cloud Pak for Dataにより、信頼できるデータへのアクセスが実現し、企業報告書のために準備すべきデータ量を削減します。ゴールデン・ソース・リポジトリーは、データ・ガバナンスを可能にし、リスク報告書およびグローバル預託証券(GDR)の手法に関するリスク・データ集計(RDA)原則へのコンプライアンス管理を支援します。包括的なデータ管理機能群は、より高速で信頼性の高いデータ消費を提供することでビジネス・ユーザー、データ・エンジニア、データサイエンティスト、およびアプリケーション開発者に、俊敏性と生産性が向上します。
Boza氏は次のように述べています。「IBM Cloudを基盤としたハイブリッドクラウド・インフラストラクチャーの開発をサポートするために、グローバル・テクノロジー・パートナーとして私たちはIBMを選択しました。これにより、より優れたサービスを提供し続け、顧客体験を向上させることができます。」

この変革は、アプリのモダナイゼーションとクラウドネイティブ開発の各段階を設定し、新しいデジタル・サービスを開発するために銀行のAPIをパートナーに公開し、充実した顧客体験を提供すると同時に、メキシコ全体の銀行の未来を再構築できるよう支援します。

Banco Sabadell S.Aについて

3年を越える運用実績を持つメキシコの複合銀行機関。国内10都市に拠点を置いています。Banco Sabadell (外部リンク)の報告書によると、法人向け銀行およびクレジットのポートフォリオは約880億メキシコペソ、資金収支は約440億メキシコ・ペソです(2020年11月30日現在)。
顧客がモバイル・バンキングを100%利用できるメキシコ初の銀行であり、メキシコのフィンテック協会に参加した最初の銀行機関でもあります。これにより、デジタル戦略が強化され、Banking as a serviceに焦点を当てた製品のパイオニアとなりました。
また、スペインで4番目に大きな総資産2,250億ユーロ以上の先進的な銀行グループBanco Sabadell Groupに属しています。Banco Sabadellは、20カ国で国際活動を行う事業分野の主要フランチャイズであり、主に欧州で2,500を超えるオフィスと約26,000人の従業員を擁しています
